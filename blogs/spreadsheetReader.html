Google Spreadsheet Reader - A jQuery Plugin

  • December 26, 2019
  • By Alexander Whipp

Multiple clients requested for backend solutions to frontend only applications. They wanted the ability to modify a spreadsheet and have the results persist to a frontend website.

Users can now create a Google Spreadsheet that has read-only permissions. From this the jQuery plugin will permit users to embed conditional logic within their front-end websites. This is a completely open source plugin. You can find the full code base on github. You can also find more indepth information on the documentation & examples page here.
